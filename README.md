# Jellyfin Remote Server Sync Plugin

This plugin allows you to synchronize users and libraries between multiple Jellyfin servers.

## Installation

1. Clone the repository.
2. Build the project using your preferred method (e.g., Visual Studio, dotnet CLI).
3. Copy the generated DLLs to the `plugins` directory of your Jellyfin server.

## Configuration

Configure the plugin through the Jellyfin dashboard under the "Plugins" section. You can add remote servers and set synchronization options.

## Usage

You can manually trigger synchronization through the API or set up automatic synchronization intervals.
